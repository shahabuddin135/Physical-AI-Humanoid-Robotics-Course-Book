# Physical AI اور ہیومنائیڈ روبوٹکس میں خوش آمدید

> **TL;DR:** آپ ایسے روبوٹس بنانا سیکھنے والے ہیں جو حقیقی دنیا میں واقعی کام کرتے ہیں۔ تیار ہو جائیں، کافی پکڑیں، اور مشین شعور کے بارے میں کچھ وجودی سوالات کے لیے تیار رہیں۔

---

## یہ کتاب کیا ہے؟

AI کا مستقبل صرف چیٹ بوٹس اور امیج جنریٹرز نہیں ہے—یہ **Physical AI** ہے: ذہین سسٹمز جو حقیقی دنیا میں موجود ہوتے ہیں اور اس کے ساتھ تعامل کرتے ہیں۔ ہم ایسے روبوٹس کی بات کر رہے ہیں جو آپ کے بکھرے ہوئے اپارٹمنٹ میں نیویگیٹ کر سکتے ہیں، اشیاء کو مینیپولیٹ کر سکتے ہیں، اور شاید (بس شاید) ہر پانچ سیکنڈ میں گریں نہیں۔

یہ کتاب اس مستقبل کی تعمیر کے لیے آپ کی رہنما ہے۔ خاص طور پر، آپ سیکھیں گے:

- 🧠 **ROS 2** — وہ اعصابی نظام جو روبوٹس کو چلاتا ہے
- 🌍 **Simulation** — جہاں آپ کا روبوٹ محفوظ طریقے سے (اور شاندار طریقے سے) ناکام ہو سکتا ہے
- ⚡ **NVIDIA Isaac** — GPU سے چلنے والی پرسیپشن اور نیویگیشن
- 🗣️ **Vision-Language-Action** — جہاں LLMs حقیقی دنیا کی روبوٹکس سے ملتے ہیں

آخر تک، آپ ایک **autonomous humanoid** بنائیں گے جو صوتی کمانڈز وصول کر سکتا ہے، اعمال کی منصوبہ بندی کر سکتا ہے، رکاوٹوں کو عبور کر سکتا ہے، اور اشیاء کو مینیپولیٹ کر سکتا ہے۔ یا کم از کم کچھ ایسا جو واقعی بہت کوشش کرتا ہے۔

---

## یہ کتاب کیوں موجود ہے؟

آئیے ایماندار بنیں: زیادہ تر روبوٹکس کے وسائل یا تو:

1.  **Academic papers** — تکنیکی طور پر درست لیکن ایسی زبان میں لکھے گئے ہیں جو آپ کو اپنی زندگی کے انتخاب پر سوال اٹھانے پر مجبور کرتی ہے
2.  **Outdated tutorials** — "سب سے پہلے، Ubuntu 16.04 پر ROS Kinetic انسٹال کریں..."
3.  **Toy examples** — "مبارک ہو! آپ کا روبوٹ دائرے میں گھوم سکتا ہے!"

یہ کتاب مختلف ہے۔ ہم نے حقیقی دنیا کی روبوٹکس کے علم کو—وہ چیز جو دراصل ہیومنائیڈ روبوٹس بنانے والی کمپنیوں میں استعمال ہوتی ہے—لیا ہے اور اسے بنایا ہے:

-   **قابل فہم** — وضاحت کے بغیر کوئی jargon نہیں
-   **عملی** — ہر باب میں عملی پروجیکٹس ہیں
-   **دلچسپ** — ڈارک مزاح شامل ہے، وجودی خوف اختیاری ہے
-   **RAG-Ready** — ہمارے AI chatbot کی مدد کے لیے منظم

## پیشگی ضروریات

شروع کرنے سے پہلے، آپ کے پاس ہونا چاہیے:

-   **Python 3.10+** — اگر آپ Python نہیں جانتے، تو وہ ایک مختلف کتاب ہے
-   **Linux (Ubuntu 22.04 تجویز کردہ)** — جی ہاں، واقعی۔ Windows والے لوگو، WSL2 آپ کا دوست ہے
-   **بنیادی terminal skills** — `cd`، `ls`، اور error messages کو Google کرنے کی صلاحیت
-   **صبر** — روبوٹکس مشکل ہے۔ اسی لیے جب یہ کام کرتا ہے تو متاثر کن ہوتا ہے

ہونا اچھا ہے:
-   ROS (کسی بھی ورژن) سے کچھ واقفیت
-   transforms اور coordinate frames کی بنیادی سمجھ
-   آپ کے کوڈ کے کام نہ کرنے پر مزاح کا احساس

## کیپ اسٹون پروجیکٹ

Module 4 تک، آپ **The Autonomous Humanoid** بنائیں گے: ایک simulated robot جو کر سکتا ہے:

1.  🎤 ایک voice command وصول کرے ("سرخ کیوب اٹھاؤ")
2.  🧠 LLM کا استعمال کرتے ہوئے action sequence کی منصوبہ بندی کرے
3.  🗺️ رکاوٹوں سے بچتے ہوئے ماحول میں Navigate کرے
4.  👁️ computer vision کا استعمال کرتے ہوئے ہدف کی چیز کی شناخت کرے
5.  🤖 چیز کو Manipulate کرے

یہ بنیادی طور پر وہی ہے جو Boston Dynamics کرتا ہے، سوائے simulation میں اور زیادہ Python کے ساتھ۔
