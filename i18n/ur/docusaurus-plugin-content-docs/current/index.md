# Physical AI اور ہیومنائیڈ روبوٹکس میں خوش آمدید

> **TL;DR:** آپ ایسے روبوٹ بنانا سیکھنے والے ہیں جو حقیقی دنیا میں کام کرتے ہیں۔ کمر کس لیں، کافی پکڑیں، اور مشین کے شعور کے بارے میں کچھ وجودی سوالات کے لیے تیار ہو جائیں۔

---

## یہ کتاب کیا ہے؟

مصنوعی ذہانت (AI) کا مستقبل صرف چیٹ بوٹس اور امیج جنریٹرز تک محدود نہیں ہے—یہ **Physical AI** ہے: ذہین نظام جو حقیقی دنیا میں موجود ہیں اور اس کے ساتھ تعامل کرتے ہیں۔ ہم ایسے روبوٹس کی بات کر رہے ہیں جو آپ کے بکھرے ہوئے اپارٹمنٹ میں نیویگیٹ کر سکتے ہیں، اشیاء کو ہینڈل کر سکتے ہیں، اور شاید (بس شاید) ہر پانچ سیکنڈ میں گر نہ جائیں۔

یہ کتاب اس مستقبل کو بنانے کے لیے آپ کی رہنما ہے۔ خاص طور پر، آپ سیکھیں گے:

- 🧠 **ROS 2** — وہ اعصابی نظام جو روبوٹس کو چلاتا ہے
- 🌍 **Simulation** — جہاں آپ کا روبوٹ محفوظ طریقے سے (اور شاندار طریقے سے) ناکام ہو سکتا ہے
- ⚡ **NVIDIA Isaac** — GPU سے چلنے والی پرسیپشن اور نیویگیشن
- 🗣️ **Vision-Language-Action** — جہاں LLMs حقیقی دنیا کی روبوٹکس سے ملتے ہیں

آخر تک، آپ ایک **خود مختار ہیومنائیڈ** بنائیں گے جو صوتی کمانڈز وصول کر سکتا ہے، اعمال کی منصوبہ بندی کر سکتا ہے، رکاوٹوں سے بچتے ہوئے نیویگیٹ کر سکتا ہے، اور اشیاء کو ہینڈل کر سکتا ہے۔ یا کم از کم کچھ ایسا جو واقعی بہت کوشش کرتا ہے۔

---

## یہ کتاب کیوں موجود ہے؟

ایمانداری سے کہیں تو: روبوٹکس کے زیادہ تر وسائل یا تو یہ ہیں:

1.  **Academic papers** — تکنیکی طور پر درست لیکن ایسی زبان میں لکھے گئے ہیں جو آپ کو اپنی زندگی کے انتخاب پر سوال اٹھانے پر مجبور کرتی ہے۔
2.  **Outdated tutorials** — "First, install ROS Kinetic on Ubuntu 16.04..."
3.  **Toy examples** — "Congratulations! Your robot can spin in a circle!"

یہ کتاب مختلف ہے۔ ہم نے حقیقی دنیا کی روبوٹکس کے علم کو — وہ چیزیں جو ہیومنائیڈ روبوٹ بنانے والی کمپنیوں میں دراصل استعمال ہوتی ہیں — لیا ہے اور اسے بنایا ہے:

-   **Understandable** — بغیر وضاحت کے کوئی اصطلاح نہیں۔
-   **Practical** — ہر باب میں عملی منصوبے ہیں۔
-   **Engaging** — تاریک مزاح شامل ہے، وجودی خوف اختیاری ہے۔
-   **RAG-Ready** — ہمارے AI چیٹ بوٹ کی مدد کے لیے منظم کیا گیا ہے۔

### 💻 کوڈ کی مثالیں

## پیشگی ضروریات

شروع کرنے سے پہلے، آپ کے پاس ہونا چاہیے:

-   **Python 3.10+** — اگر آپ Python نہیں جانتے، تو وہ ایک مختلف کتاب ہے۔
-   **Linux (Ubuntu 22.04 recommended)** — ہاں، واقعی۔ Windows استعمال کرنے والو، WSL2 آپ کا دوست ہے۔
-   **Basic terminal skills** — `cd`، `ls`، اور غلطی کے پیغامات کو Google کرنے کی صلاحیت
-   **Patience** — روبوٹکس مشکل ہے۔ اسی لیے جب یہ کام کرتا ہے تو متاثر کن ہوتا ہے۔

ہونا اچھا ہے:
-   ROS (کسی بھی ورژن) سے کچھ واقفیت
-   ٹرانسفارمز اور کوآرڈینیٹ فریمز کی بنیادی سمجھ
-   آپ کے کوڈ کے کام نہ کرنے کے بارے میں مزاح کا احساس

## کیپ اسٹون پروجیکٹ

ماڈیول 4 تک، آپ **خود مختار ہیومنائیڈ** بنائیں گے: ایک سمیولیٹڈ روبوٹ جو کر سکتا ہے:

1.  🎤 Receive a voice command ("Pick up the red cube")
2.  🧠 Use an LLM to plan the action sequence
3.  🗺️ Navigate through the environment avoiding obstacles
4.  👁️ Identify the target object using computer vision
5.  🤖 Manipulate the object

یہ بنیادی طور پر وہی ہے جو Boston Dynamics کرتا ہے، سوائے اس کے کہ یہ سمولیشن میں اور زیادہ Python کے ساتھ ہے۔
